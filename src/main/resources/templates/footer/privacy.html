Cookies policy

  1. For your convenience our Website might be using Cookies files and similar technologies, in order to align the services we provide you. Cookies are small text files sent by the visited website to the users device.

  2. Cookies files in our Website might be used to:
    • Adjust the website to the user preferences (e.g. to recognize the User's device to show the website properly),
    • Create statistics that help understanding how Users use the website, which helps enhancing the Website,
    • Keep User's session (after logging in), which allows the User to stay logged in in the subpages of the Website.
  3. Our Website might be using the two basic types of Cookies:
    • session - temporary files, which are kept on the User's device until he logs out, leaves the website or closes the web browser,
    • persistent - files kept on the user's device for a fixed period of time (until expiration date) or until the User removes them.
  4. Our Website might be using the following types of Cookies:
    • Essential Cookies - allowing usage of the services available in the Website, e.g. authorization cookies,
    • Security Cookies - used to provide security features, e.g. to detect illegal usage of the Website's services,
    • Performance Cookies - monitoring site performance and the way it is used,
    • Functional Cookies - allowing remembering of user preferences and settings.

  5. In many cases the software used to browse the websites (Web Browser) by default allows the Cookies files in the User's end device. The Website's Users can modify the Cookies files settings at any time. The settings can be modified especially in the way, so that the automatic handling of Cookies was blocked in the Web Browser's settings or inform about each of putting them in the Website's User device.

  6. Restrictions in usage of Cookies files might influence on some of the functionalities available on the pages of the Website.

  7. In order not to receive Cookies files, you should change your Web Browser's settings to automatically revoke Cookies. The browser's settings usually also allow you to manage Cookies in other extent.
