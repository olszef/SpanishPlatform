Time

In Spain on a daily basis we typically use 12-hour system. However, we can also use the full 24-hour, but it is usually used only in offical communication, like for example in different schedules.

To talk about time we need to construct the information from a few parts. It would be also good to know how to ask for the time. Have a look below to find all that you need.

What time is it?

To ask for time we can use the following question:

  • ¿Qué hora es?
  • ¿Tiene hora?

Talk about hours

Sentence construction

Talking about hours we use:

  1. the verb SER in only two forms:
    • es - when talking about the hour "one"
    • son - when talking about all other hours
  2. definite articles in feminine gender before the numeral:
    • la - when talking about the hour "one"
    • las - when talking about all other hours
  3. cardinal numeral in feminine gender to represent the hour:
    • una / dos /très / etc.
  4. conjunctions to talk about minutes:
    • current hour + en punto + number - when it is exactly the given hour
    • current hour + y + number - to talk about minutes from 1 to 30
    • next hour + menos + number - to talk about minutes from 31 to 59
  5. part of the day if the context doesn't allow to define it:
    • de la mañana - in the morning
    • del mediodía - at noon
    • de la tarde - in the afternoon
    • de la noche - at night
    • del medianoche - at midnight

Examples

Following are some examples:

  • 13:13 - Es la una y trece de la tarde.
  • 08:56 - Son las nueve menos cuatro de la mañana.
  • 17:00 - Son las cinco en punto.
